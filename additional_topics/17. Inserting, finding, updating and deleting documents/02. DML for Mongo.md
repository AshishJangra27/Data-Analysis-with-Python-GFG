# MongoDB DML (Data Manipulation Language) Operations

## INSERT Operations

### insertOne() - Insert Single Document
```javascript


// Insert a single document into users collection
db.users.insertOne({ name: "Alice", email: "alice@example.com", age: 30 })

// Additional example
db.users.insertOne({ name: "Eve", email: "eve@example.com", age: 27 })
```

### insertMany() - Insert Multiple Documents
```javascript
// Insert multiple documents at once
db.users.insertMany([
  { name: "Bob", email: "bob@example.com", age: 28 }, 
  { name: "Carol", email: "carol@example.com", age: 35 }
])

// Additional example
db.users.insertMany([
  { name: "Frank", email: "frank@example.com", age: 40 },
  { name: "Grace", email: "grace@example.com", age: 32 }
])
```

## READ/FIND Operations

### Basic Find - Retrieve All Documents
```javascript
// Get all documents from collection
db.users.find()
```

### Find with Comparison Operators
```javascript
// Less than
db.users.find({ age: { $lt: 25 } })

// Less than or equal
db.users.find({ age: { $lte: 25 } })

// Greater than
db.users.find({ age: { $gt: 25 } })

// Greater than or equal
db.users.find({ age: { $gte: 25 } })

// Equal
db.users.find({ age: { $eq: 25 } })
```

### Find with Projection - Select Specific Fields
```javascript
// Get all documents but only show name field (exclude _id)
db.users.find({}, { name: 1, _id: 0 })

// Find with filter AND projection
db.users.find({ age: { $gt: 38 } }, { name: 1, _id: 0 })

// Multiple field projections
db.users.find({ age: { $gt: 25 } }, { name: 1, _id: 0, age: 1 })

// Additional example
db.users.find({ age: { $gte: 25 } }, { name: 1, email: 1, _id: 0 })
```

### findOne() - Get Single Document
```javascript
// Find first document matching criteria
db.users.findOne({ email: "alice@example.com" })

// Additional example
db.users.findOne({ name: "Bob" })
```

### countDocuments() - Count Matching Documents
```javascript
// Count documents matching criteria
db.users.countDocuments({ age: { $gte: 30 } })

// Additional example
db.users.countDocuments({ age: { $lt: 30 } })
```

## UPDATE Operations

### updateOne() - Update Single Document
```javascript
// Update first matching document
db.users.updateOne({ name: "Alice" }, { $set: { age: 31 } })

// Verify the update
db.users.find({ name: { $eq: "Alice" } }, { name: 1, _id: 0, age: 1 })

// Additional example
db.users.updateOne({ email: "bob@example.com" }, { $set: { age: 29 } })
```

### updateMany() - Update Multiple Documents
```javascript
// Update all matching documents
db.users.updateMany({ name: "Alice" }, { $set: { names: "Alison" } })

// Additional example
db.users.updateMany({ age: { $gte: 30 } }, { $set: { senior: true } })
```

## BULK WRITE Operations

### bulkWrite() - Execute Multiple Operations
```javascript
// Execute multiple insert, update, and delete operations
db.users.bulkWrite([
  { insertOne: { document: { name: "Dan", email: "dan@example.com" } } },
  { updateOne: { filter: { name: "Carol" }, update: { $set: { age: 36 } } } },
  { deleteOne: { filter: { name: "Obsolete" } } }
])

// Additional example
db.users.bulkWrite([
  { insertOne: { document: { name: "Henry", email: "henry@example.com", age: 45 } } },
  { updateOne: { filter: { name: "Eve" }, update: { $set: { email: "eve.new@example.com" } } }
])

// Verify results
db.users.find()
```
