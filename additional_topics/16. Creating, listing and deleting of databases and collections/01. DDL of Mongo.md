# MongoDB DDL Operations

## Server Management
```bash
# Start local MongoDB service
brew services start mongodb-community

# Stop MongoDB service
brew services stop mongodb-community
```

## Connect to MongoDB
```bash
# Open local MongoDB shell
mongosh

# Connect to remote MongoDB Atlas cluster
mongosh "mongodb+srv://ashishjangraelite_db_user:ashishjangra@db.igztzq7.mongodb.net/"
```

## Database Operations
```javascript
// List all databases
show dbs

// Get current database
db

// Switch/Create database
use swiggy

// Get current database name
db.getName()

// Get database statistics
db.stats()
```

## Collection Operations
```javascript
// List all collections in current database
show collections

// Get collection names as array
db.getCollectionNames()

// Create a new collection
db.createCollection("users")

// Drop/Delete a collection
db.users.drop()

// Verify collection deletion
db.getCollectionNames()
```
